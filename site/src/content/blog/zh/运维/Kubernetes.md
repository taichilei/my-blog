**Kubernetes**

**K8s** 是**Kubernetes**的简称，本质上是一个**容器编排系统**，用于在多台服务器上**自动化部署、管理和扩展容器化应用**。

K8s 专注解决四件事：

​	1.	**调度**：容器应该跑在哪台机器上

​	2.	**自愈**：容器挂了自动重启、迁移

​	3.	**伸缩**：根据负载自动扩容/缩容

​	4.	**发布**：滚动更新、版本回退不中断服务

核心概念

**Pod：最小调度单位（不是容器本身）**

​	•	**Node**：一台工作机器（物理机或虚拟机）

​	•	**Cluster**：一组 Node

​	•	**Service**：稳定访问入口，解决 Pod IP 变化问题

​	•	**Deployment**：声明“我想要多少副本、什么版本”

**典型应用场景**

​	•	微服务后端（Spring Boot / Go 服务）

​	•	高并发 Web 服务

​	•	CI/CD 自动部署

​	•	云原生架构（Cloud Native）
